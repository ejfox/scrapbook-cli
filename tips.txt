# Tips & Recipes

Quick reference for common scrapbook-cli workflows.

## Interactive Search

Browse and select bookmarks with fuzzy finding:

```bash
scrap search "python" --fzf | fzf
scrap list --fzf | fzf
```

## Extract Data

```bash
# All URLs
scrap list --json | jq -r '.[].url'

# All titles
scrap list --json | jq -r '.[].title'

# Random bookmark
scrap list --json | jq -r '.[].url' | shuf -n 1
```

## Count & Statistics

```bash
# Total bookmarks
scrap list --json | jq 'length'

# Bookmarks by source
scrap list --json | jq 'group_by(.source) | map({source: .[0].source, count: length})'
```

## Tags

```bash
# All unique tags
scrap list --json | jq -r '.[].tags[]' | sort | uniq

# Most common tags
scrap list --json | jq -r '.[].tags[]' | sort | uniq -c | sort -rn | head -10
```

## Entity Knowledge Graph

```bash
# Count mentions of an entity
scrap entity "OpenAI" --json | jq '.total_scraps'

# Get all titles mentioning entity
scrap entity "Claude" --json | jq -r '.scraps[].title'
```

## Export & Backup

```bash
# Full JSON backup
scrap list --json > backup.json

# JSONL (one per line, version control friendly)
scrap list --jsonl > backup.jsonl

# CSV for spreadsheets
scrap list --csv > bookmarks.csv
```

## Analyze with LLM

Pipe extracted data to CLI LLM tools (`llm`, `gpt-cli`, etc):

```bash
# Summarize search results
scrap search "ai" --json | jq -r '.[] | .title' | llm "Summarize these topics"

# Extract themes from summaries
scrap list --json --limit 5 | jq -r '.[].summary' | llm "What are the main themes?"

# Analyze entity mentions
scrap entity "Tesla" --json | jq -r '.scraps[].title' | llm "What topics are covered?"
```

## Development

```bash
# Link to local dev version
npm link

# Create short alias
alias scrap="scrapbook-cli"

# All changes are live
scrap search "test"
```
